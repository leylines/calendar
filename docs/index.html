    CSV Format Info

    Du kannst eigene Termine über eine .csv Datei einlesen. Diese sollte wie folgt formatiert sein:

    Wiederkehrend (jährlich):
    05-18, Geburtstag Sirija
    12-24, Weihnachten
    Einmalig (bestimmtes Jahr):
    2026-06-15, Zahnarzt Termin
    2026-10-31, Halloween Party

    Mehrere Termine pro Tag sind möglich (neue Zeile in der CSV).
